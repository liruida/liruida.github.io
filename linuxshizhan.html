1、文件操作命令上 touch file cp mv rm

改变命令行背景颜色，右击，changs settings colour modty

touch 新建空文件夹或更新文件时间的标记

file 查看文件类型

touch file1.txt file2.txt(同时创建了两个文件)

is查看所有文件，发现目录蓝色和文件黑色颜色不同

file 1 输出 1：directory(目录)

cp file1 file2.txt 1 将file1 file2.txt复制到1中，因为file1 file2.txt与1在同目录下，故可以使用相对路径

cd 1 进入目录1

Ls 展示目录1下的所有内容

pwd 查看当前所在路径

cd ..回到上一级目录

clear 清屏

cp -r 1 2 因为1不是空的 -r是递归复制整个目录树

ls -l 查看目录下所有文件及目录属性

cp -p file1 3 保持file1复制过去后属性不变

rm file1 删除file1

rm -r 1 递归删除整个目录树

mkdir 1 建立空白目录1

mv 1 3 5 将目录1 3 移动到5中

2、文件操作命令下 which find ln

echo $PATH echo意为输出 $PATH指环境变量 which只能在环境变量范围内查找内置命令文件和显示路径，而找不到shell的内部命令，例如cd。

which mkdir 输出/bin/madir

find /-name file1.txt find 寻找文件或目录，/表示当前根目录，-name 表示按文件名称查加

Ctrl+C结束命令

find 查找条件 范围

查找条件 -name 按名称查找 -type 按文件类型查找 -size按文件大小查找 -user按文件属主查找

find -type directory

ln -s /home/file1.txt /home/2 在绝对路径目录/home/2下建立一个软链接文件file1.txt写上-S便是软链接，/home/file1.txt便是母文件， /home/2/file1.txt便是子文件，当子文件修改时母文件也会随之修改，当母文件修改时子文件也会随之修改。而删除子文件时母文件不变，而删除母文件时子文件随之毁损。 不写-S便是硬链接，目录不能建立硬链接，其余与软链接相同。

3、对文件内容操作命令

cat file1.txt 显示出文件内容操作系统

cat file1.txt | more 全屏方式分页显示文件内容，交互操作方法：按enter键向下逐行滚动，按空格键向下翻一屏，按B键向上翻一屏，按Q键退出。

cat file1.txt | less 与more相同

head -3 file1.txt 查看文件开头的一部分内容，-n控制行数，默认为10行

tail -3 file1.txt 查看文件末尾的一部分内容，-n控制行数，默认为10行

wc -l file1.txt 统计行数

wc -w file1.txt 统计单词个数

wc -c file1.txt 统字字节数

grep:在文件中查找并显示包含指定字符串的行

文档中带#开头的是注释

grep -v "^#" file1.txt "^"表示以...开头 -v表示输出不含有所查找的内容，此条命令的含义就是输出去掉文档里的所有注释的内容。

grep -v "^$" file1.txt "^$"表示空行 此条命令的含义就是输出去掉文档里的所有含有空行的内容。

grep -v "^$" file1.txt | grep -v "^#" file1.txt 输出不含有空格和# |表示通道命令，是指grep -v "^$" file1.txt的结果输出 作为 grep -v "^#" file1.txt 执行前的输入
